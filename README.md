# 🎥 Fact Check Video

A web-based application that helps users verify the authenticity of video content by analyzing uploaded videos and providing fact-checking insights. The platform combines a modern frontend interface with a Python-powered backend for processing and analysis.

## 🚀 Features

- Upload and analyze video content
- User-friendly interface
- Fast video processing
- Fact-checking workflow
- Frontend and backend separation for scalability
- REST API communication between client and server

## 🛠️ Tech Stack

### Frontend
- TypeScript
- Vite
- HTML
- CSS
- JavaScript

### Backend
- Python

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/fact-check-video.git
cd fact-check-video
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm start
```
